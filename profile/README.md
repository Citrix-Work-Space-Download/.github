# Citrix Workspace Access

**Citrix Workspace Access** is a secure unified platform that delivers virtual applications, desktops, and corporate data to any device. As a comprehensive **Citrix Workspace Solution**, it aggregates local, web, and SaaS applications into a single interface with contextual security policies. This **Citrix Workspace Client** optimizes remote productivity through adaptive compression and protocol acceleration. With **Citrix Workspace Hub**, you can connect to published resources, access virtualized business tools, and maintain endpoint compliance without storing sensitive data locally. The platform includes single sign-on and performance analytics.

## Features

- **Unified Workspace**: The **Citrix Workspace Access** aggregates all work resources in one location.
- **Virtual App Delivery**: Use **Citrix Workspace Solution** to stream applications without local installation.
- **Contextual Security**: As a **Citrix Workspace Client**, it applies policies based on user, device, and location.
- **Performance Optimization**: This **Citrix Workspace Hub** uses HDX protocol for responsive sessions.
